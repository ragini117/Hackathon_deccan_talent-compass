# Talent Compass

Talent Compass is an AI-powered recruitment tool designed to streamline the hiring process. It leverages OpenAI's GPT-4o-mini to parse job descriptions, rank candidates, and simulate recruiter-candidate conversations.

## Features
- **Job Description Parsing**: Extracts fields like role, experience, location, education, and salary from structured and unstructured formats.
- **Candidate Ranking**: Scores candidates based on skill, experience, and domain match.
- **Recruiter Simulation**: Generates realistic recruiter-candidate conversations to assess interest and expectations.

## Technologies Used
- **Backend**: Node.js, Express.js, OpenAI API
- **Frontend**: React, TypeScript, Tailwind CSS
- **Database**: Supabase
- **Testing**: Vitest
